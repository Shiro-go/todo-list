# ✨ Neon Todo App

A minimalist browser-based todo application with a **neon blue glass UI**, built using **pure HTML, CSS, and JavaScript**.

No frameworks. No backend. Just clean frontend logic and local persistence.

---

## 🚀 Features

* Add tasks with:

  * Title
  * Description
  * State (Not done / In progress / Finished)

* Task list:

  * Indexed order display
  * Clean layout with emphasis on readability
  * Inline state switching
  * Delete tasks individually

* Filters:

  * Toggle visibility by state
  * Color-coded (grey / orange / green)

* Sorting:

  * By creation order
  * By state
  * By title

* Storage:

  * Uses **localStorage**
  * Tasks persist after refresh
  * Works perfectly on GitHub Pages
